## CDPSfy

### Dependencies

- node.js
- npm

### How to install

<pre>
git clone https://github.com/sirasistant/CDPSfy.git
cd CDPSfy
npm install
</pre>

### How to configure

+ The server is configured to listen in port 8080. You can modify that folder using the env variable 'PORT'

### How to run

<pre>
npm start
</pre>

#### Default music: http://www.bensound.com
